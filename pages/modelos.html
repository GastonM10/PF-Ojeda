Conoce nuestros modelos

Casas simples

Una casa contenedor simple, es una vivienda construída utilizando contenedores de transporte como base estructural. Suele ser compacta y funcional, con espacios bien definidos y eficientes, aprovechando la geometría del contenedor.

Casas dobles

Una casa contenedor doble, es una vivienda que utiliza dos contenedores de transporte interconectados para crear un espacio habitable más amplio. Esta configuración permite diseñar una casa con una distribución más espaciosa y versátil, manteniendo la base estructural de los contenedores.
